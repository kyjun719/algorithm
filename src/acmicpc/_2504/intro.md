input</br>
```
(()[[]])([])

[][]((])
```

```
28

```



#### 재정의와 추상화<br>
-- 문제를 읽고 본인이 이해하기 쉬운 방식으로 해석<br>
(X)는 2X, [X]는 3X이고 XY는 X+Y일때 전체 계산값
